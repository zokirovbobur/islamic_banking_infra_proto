Maqsad
Murakkab ko'p bosqichli bank jarayonlarini avtomatlashtiradi — financing ariza qabul qilishdan to disbursement gacha har bir qadam tartibli bajariladi. Saga pattern orqali distributed transaction xatolarini ham boshqaradi.
Asosiy xususiyatlar
  • BPMN 2.0 standart — vizual jarayon dizayni
  • Saga pattern — distributed transaction rollback
  • Financing workflow: ariza → KYC → Sharia check → tasdiq → disbursement
  • Parallel gateway — bir vaqtda bir necha approval oqimi
  • Timer event — muddat o'tsa avtomatik eslatma yoki bekor qilish
  • Audit trail — har bir qadam vaqt tamg'asi bilan saqlanadi
  • REST API orqali tashqi tizimlar bilan integratsiya
  • Low-code modelling — IT bo'lmagan xodimlar ham jarayon chiza oladi
